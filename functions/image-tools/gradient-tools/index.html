Gradient Generator

Create multi-layer gradients and bake them into a single image. Configuration is automatically saved in PNG metadata.

Instructions
  • Add/Remove Gradient Rows: Click the + button at the bottom to add a new row, click the X on the right of each row to remove it.
  • Sort Gradient Rows: Click and drag gradient rows to sort them.
  • Add Color Stop: Hold Ctrl and click anywhere on the gradient bar.
  • Remove Color Stop: Hold Ctrl and click on an existing color stop.
  • Change Color: Click on a color stop to open the color picker.
  • Move Color Stop: Click and drag a color stop left or right.
  • Load Configuration: Drag and drop a previously exported gradient.png file into the dashed box on the right.
